# Credits

## OpenSpec

Specwork's spec system is based on [OpenSpec](https://github.com/Fission-AI/OpenSpec) by [Fission AI](https://github.com/Fission-AI).

The spec conventions (proposal, design, tasks workflow, GIVEN/WHEN/THEN scenario format, delta spec system) were adapted from OpenSpec and integrated into Specwork as a built-in feature.

We thank the OpenSpec team for their foundational work on spec-driven development.

## Claude Code

Specwork is built on [Claude Code](https://docs.anthropic.com/en/docs/claude-code) by [Anthropic](https://anthropic.com). The skills, agents, hooks, and commands system that powers Specwork's execution engine is provided by Claude Code's extension architecture.
